Batch Update

(Update certificates, user accounts, network configurations)

 

 

Find more recent updates at update log.

 

If you are building a new system or you are not sure about your system status, follow the steps below and do the full-update:

 

1.      Setup certificates and updates

2.      Open firewall for users/hosts/ips

3.      Update user accounts (either individually or in batch, using pragma_setuser script)

a.      Update in batch

1.      download this package (includes all users info packages and pragma_setuser script) (check md5sum)

2.      unzip and untar the updaccounts.tar.gz file

3.      modify updaccounts.txt file to remove lines for any users who already have accounts on your system

4.      change updaccounts.txt file to a executable shell script

5.      run the shell script under root, answer all questions to install all users’ accounts

b.      Update invividually

1.      download the users info pack (check md5sum)

2.      download the pragma_setuser script

3.      login an account with root priviledge

4.      cd to the directory where pragma_setuser and the user’s info pack is

5.      make sure that pragma_setuser is executable (chmod +x pragma_setuser)

6.      run the script with the user’s info pack file. For example, if user’s info pack is zhengc.tar.gz, run

# ./pragma_setuser zhengc.tar.gz

then answer all the questions.

4.      Retire user accounts

See the list of retired users info packs and remove them from your system.
